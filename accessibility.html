Accessibility Statement

Last updated: March 2026

Our Commitment

Life Built in Kentucky is committed to ensuring this website is accessible to all visitors, including those with disabilities. We work to meet or exceed the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards.

Features

  • Skip navigation links for keyboard users
  • Semantic HTML structure with proper heading hierarchy
  • Descriptive alt text for all meaningful images
  • Sufficient color contrast throughout the site
  • Keyboard-navigable menus and interactive elements
  • ARIA labels on navigation and interactive components

Known Limitations

We are continually working to improve accessibility. If you encounter any barriers, please contact us and we will work to address them promptly.
